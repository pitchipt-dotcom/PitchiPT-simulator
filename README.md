# PitchiPT-simulator

理学療法士（PT）を目指す学生や、若手療法士のための臨床シミュレーター（ケーススタディツール）です。教科書通りの知識だけでなく、実際のベッドサイドで必要となる「臨床思考プロセス」を疑似体験し、リスク管理やゴール設定の判断力を養うことを目的としています。

## 開発の背景と目的（問題意識）
多くの若手療法士や学生は、「教科書の知識はあっても、実際の患者さんを前にすると何を優先すべきか分からない」「ベッドサイドでのリスク管理に不安がある」という悩みを抱えています。
本ツールは、そうした「教科書と臨床のギャップ」を埋め、臨床での思考を整理するためのスパーリングパートナーとして開発しました。

## 特徴・学べること
- **リアルな臨床シナリオ**: 実際のベッドサイドを想定した事例（ケース）をもとに進行します。
- **臨床思考の体系化**: 疾患の理解、リスク管理、ゴール設定までの一連の流れを整理できます。
- **実践的な判断力の育成**: 選択肢や状況の変化に応じたシミュレーションが可能です。
